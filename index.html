Git commands

  • git init

    initializes the connection to linked github account

  • git add .

    indicates a following commit entry will update a particular file

  • git commit -m "message"

    use commit name to describe the updates in this new version or to name the version more generally

  • initiate link to repository

    *the above is not command text*: copy three lines from box labelled "…or push an existing repository from the command line" in the new repository and paste in Terminal, hit return

  • git push

    use to publish local changes to the repository using the following sequence of commands:

    • git add .
    • git commit -m "describe changes"
    • git push
